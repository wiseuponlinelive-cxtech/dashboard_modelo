Como ler o painel
Guia rápido para qualquer pessoa da operação.

Este painel mostra, de forma automática, o que aconteceu com cada conversa/aluno no período — sem precisar abrir conversa por conversa no Chatwoot. Você escolhe os filtros no topo, lê os números e, quando precisa saber quem são os alunos por trás de um número, vai para a página Operação.

As duas páginas

▦ Dashboard — visão geral

Responde o que aconteceu: os números do período em cards com quantidade e %, gráficos de performance e tabelas de resumo por status, campanha e falhas por motivo. O resumo por caixa fica na página Operação para não poluir a leitura geral. Use para ter o panorama rápido.

▤ Operação — visão analítica

Responde quem são os alunos que compõem os números. Traz a lista analítica (data de entrada, entrada ativo/reativo, nome, telefone, e-mail, ID da conversa, caixa, campanha, status, etiquetas e tabulação) que respeita os filtros e pode ser exportada em CSV.

Como usar os filtros

  1. Selecione o Período. Use os presets ou escolha Personalizado; ao selecionar essa opção, as datas de início e fim aparecem dentro do próprio controle de período para escolher qualquer intervalo.
  2. Comece pela Caixa — é onde a conversa está no Chatwoot. Nem toda conversa tem campanha, então a caixa vem primeiro.
  3. Se quiser, adicione a Campanha (o disparo/régua de origem) para olhar só um envio específico.
  4. Refine com Etiqueta / Tabulação e Status. Na página Operação, use também Agente, Time e Origem. Os filtros Campanha, Status, Etiqueta / Tabulação e Falha de envio permitem selecionar uma, várias ou todas as opções.
  5. Leia os cards e tabelas. Para ver os alunos, vá para Operação; para baixar a lista, clique em Exportar lista filtrada CSV. A exportação respeita os filtros aplicados.

Como ler gráficos e percentuais

Os percentuais são recalculados automaticamente conforme os filtros aplicados. Assim, ao escolher uma caixa, campanha, etiqueta/tabulação ou período, os cards, gráficos e tabelas passam a mostrar a performance apenas daquele recorte.

Como ler as etiquetas

As etiquetas aparecem com a origem de aplicação para diferenciar o que foi classificado automaticamente pelo fluxo/IA e o que foi ajustado manualmente por um colaborador.

MarcaçãoOrigemComo interpretar
sem_retornoEtiqueta/TabulaçãoClassificação usada para filtrar e agrupar conversas.
transbordoEtiqueta/TabulaçãoClassificação usada para identificar conversas direcionadas para outro atendimento.

Como interpretar etiquetas e tabulações

Etiqueta mostra uma marcação da conversa, podendo vir do fluxo/IA ou de um colaborador. Tabulação mostra o assunto ou resultado registrado no atendimento, como pagou, link de pagamento enviado, sem retorno, cancelamento solicitado ou AZ agendada.

Para não depender de uma medição complexa de “objetivo realizado”, o painel junta Etiqueta / Tabulação em um único filtro. Assim, a operação consegue selecionar uma ou várias classificações e exportar todos os alunos relacionados.

Caixa não é campanha

Caixa é onde a conversa entrou/está no Chatwoot. Campanha é o disparo/régua que originou a conversa. A mesma caixa pode receber conversas de campanha e por demanda espontânea (quando o aluno chega por conta própria, sem disparo).
